# 明天穿啥

**明天穿啥**，你的衣物管理和穿搭小助手！

基于react-native开发的衣物管理APP，所有数据均使用本地存储，所有功能均可离线使用。

## 主要功能

- 衣物
  - 录入衣物数据
  - 根据最近使用时间、使用次数和购买时间管理衣物
- 穿搭
  - 记录每日穿搭
  - 关联穿搭和衣物信息
